#### Linkable (i.e. "content" words)

```ts
<TextSegmentComponent
  textSegment={{
    text: 'ἀκοῦσαι',
    color: 0,
    type: 'source',
    position: 1,
  }}
/>
```

#### Non-Linkable / Disabled (i.e. "function" words)

```ts
<TextSegmentComponent
  textSegment={{
    text: 'ἀκοῦσαι',
    color: 0,
    type: 'source',
    position: 1,
  }}
  isDisabled
/>
```
